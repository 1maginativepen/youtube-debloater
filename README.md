Installation steps:

Download and unzip yt-cleaner.zip
Open Chrome and go to chrome://extensions
Enable Developer mode (top-right toggle)
Click "Load unpacked" and select the yt-cleaner folder
The extension icon will appear in your toolbar

What it does:

🚫 Block Shorts — hides the Shorts shelf everywhere (home, search, sidebar nav)
🎯 Block Recommendations — removes the right-side sidebar, related videos, and end-screen cards so nothing distracts you mid-video
⚡ Block Ads — auto-clicks skip buttons, seeks through unskippable ads, and hides banner/overlay ads
